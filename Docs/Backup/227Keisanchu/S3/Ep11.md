今回はテーマに沿って滝川＆小宮＆他のメンバー１人が作品を制作。<br>

相田能否從中猜出滝川作品的問答將會舉行！<br>
如果揀選了滝川以外的作品﹐「理解者」的稱號將會降格⁉︎<br>
