#### 關於 About
これは、22/7メンバー成長のためのプロジェクトーー<br>
メンバーそれぞれが六番町学院の後輩を<br>
指導・育成することになる。<br><br>
このプロジェクトに参加する後輩を<br>
決定するのは皆様の投票。<br><br>
皆様の手によって運命が大きく変わるプロジェクト。<br>
<blockquote>
這是個關於22/7成員成長的計劃─<br>
每位成員將在六番町学院<br>
指導、培訓後輩。<br><br>
參加這個計劃的後輩<br>
將由各位的投票決定。<br><br>
一個因各位的手而命運產生巨變的計劃。<br>
</blockquote>

#### 規則 Rule
アプリゲーム「22/7 音楽の時間」における新プロジェクトが始動いたします。<br>
このプロジェクトでは皆様の投票によって、30キャラクターの中から「22/7」とゲーム内で<br>
先輩後輩関係になる11キャラが選ばれます。<br><br>
投票は1日で「1キャラに1票」、最大で3キャラまで投票することが可能です。<br>
<blockquote>
手遊「22/7 音楽の時間」新的計劃即將開始。<br>
在這個計劃中的30名角色內，將由您投票選出11名並在「22/7」遊戲內有前後輩關係。<br><br>
每日可投「一個角色一票」，每日最多可投三票。<br>
</blockquote>
